# Cost-Estimation
Cost estimation for deploying a low-cost web application in a testing environment using a single EC2 instance (t3.micro) with minimal storage and usage


- Monthly Cost: ~$10.07 (~₹800–₹850)
- Yearly Cost: ~$120.84 (~₹10,000)

> Calculated using AWS Pricing Calculator for a t2.micro instance (730 hours/month), 8GB EBS storage, and minimal data transfer (1GB).
