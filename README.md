## 💰 Detailed Cost Breakdown (AWS Pricing Calculator)

 🖥️ EC2 Instance (t2.micro)
- Instance Type: t2.micro (1 vCPU, 1 GB RAM)
- Usage: 730 hours/month
- Cost: $0.0124/hour

➡️ Monthly Cost:
= 0.0124 × 730  
= **$9.05/month**

- Price: $0.114 per GB/month

➡️ Monthly Cost:

= **$0.91/month**

### 🌐 Data Transfer (Outbound - Internet)

### 📊 Total Monthly Cost
